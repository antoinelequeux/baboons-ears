How to use it offline

  1. Put this folder on GitHub Pages.
  2. Open the site once while online.
  3. Add it to the phone home screen.
  4. Open it again from the icon before leaving for the field.

The service worker caches the app shell, the JSON dataset, and the ear photos. The first launch may take a moment while images are stored locally.

iPhone can remove cached data when storage is tight, so test the installed app on the actual device before relying on it in the field.
